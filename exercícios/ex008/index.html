Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica)

Nesta outra frase também temos um termo em negrito, porém utilizando a tag STRONG (semântica)

Itálico

Nesta frase temos um termo em itálico usando a tag I.

Nesta outra, temos um termo também em itálico, porém com a tag EM (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK

Neste paragráfo há outro texto marcado no final do texto.

Texto grande e pequeno

Podemos formatar um texto adicionando palavras grande ou palavras pequenas, usando as tags BIG (obsoleta) ou SMALL.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas desconsiderado.

Texto inserido

Para dar ênfase a um termo ou frase num texto, usa-se a tag INS para indicar que foi adicionado depois.

Também pode usar a tag U para sublinhar o termo, mas é não semântico.

Texto sobrescrito

É utilizado com termos como: x20+3 com a tag SUP

Texto subscrito

Usado para representações como: H20 com a tag SUB
